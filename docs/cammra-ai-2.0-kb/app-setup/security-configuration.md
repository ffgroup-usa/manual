# Security Configuration

## Security/HTTPS Settings

![Security Settings](./img/security-config-1.png)

To enhance the security of your CAMMRA AI application, you have the option to enable HTTPS encryption and use self-signed certificates.

By turning on HTTPS encryption and utilizing self-signed certificates, you can ensure that the communication between the application and browsers is secure and protected. This helps safeguard the integrity and confidentiality of data transmission.

**Configuration Options:**

- **HTTPS:** Toggle On/Off to enable or disable HTTPS encryption
- **Signed Type:** Choose between:
  - **Self-signed:** Use a self-signed certificate
  - **CA Signed:** Use a certificate signed by a Certificate Authority
