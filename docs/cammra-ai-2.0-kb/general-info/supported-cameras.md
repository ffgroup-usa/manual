## Recommended Cameras

- [AXIS Q1800-LE](https://www.axis.com/products/axis-q1800-le) (Top choice)
- [AXIS Q1805-LE](https://www.axis.com/products/axis-q1805-le) (Recommended for longer distance)
- [AXIS P1465-LE 29 mm](https://www.axis.com/products/axis-p1465-le) (Recommended for shorter distance)
- [AXIS P3265-LVE 22 mm](https://www.axis.com/products/axis-p3265-lve) (Recommended for shorter distance)
- [AXIS Q1686-DLE Radar-Video Fusion Camera](https://www.axis.com/products/axis-q1686-dle) (Recommended for longer distance)
- [AXIS Q1656-DLE Fusion camera](https://www.axis.com/products/axis-q1656-dle) (Recommended for shorter distance)

## Other Supported Cameras

- P1465-LE 9mm
- P3265-LVE 9mm
- P3267-LVE
- P3268-LVE
- P1467-LE
- P1468-LE
- Q1806-LE
- Q1808-LE
- P1518-LE

:::note
CAMMRA AI works with default camera sensor #1 as video source, except AXIS P1518-E and P1518-LE cameras, where sensor #1 is used for wide-angle scene view, so CAMMRA AI utilizes sensor #2.
:::
